# Contributing to MediaSniffer Pro

Thank you for taking the time to contribute to MediaSniffer Pro!

## Issues

- Before raising an issue, please check if it has already been reported or addressed.
- Clearly describe the problem or feature request, including steps to reproduce and the expected behaviour.
- Label the issue appropriately (bug, enhancement, documentation, etc.).

## Pull Requests

- Fork the repository and create your feature branch from `main`.
- Write clear, concise commit messages.
- Follow [PEP 8](https://www.python.org/dev/peps/pep-0008/) coding style.
- Include tests if you are adding new features or fixing bugs.
- Update the documentation as appropriate.
- Submit your pull request and describe your changes.

## Code of Conduct

This project has a friendly community. Please be respectful and considerate when communicating with others. Harassment or discrimination will not be tolerated.
