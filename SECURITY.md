# Security Policy

If you discover a security vulnerability, please open an issue or contact us at abubakarsalihawa019@gmail.com.

## Supported Versions

We currently support the latest version of MediaSniffer Pro. Please update to the most recent version before reporting vulnerabilities.

## Reporting a Vulnerability

Please include a detailed description of the vulnerability with steps to reproduce and any potential impact. We will respond as quickly as possible and take appropriate action to address the issue.
